A modern, attractive **Learning Management System (LMS) organization console** built with:

- React (Vite)
- JavaScript (ES6+)
- Tailwind CSS
- Ready to integrate with Node.js, Express.js, and MongoDB APIs

## Getting Started

1. **Install dependencies**

```bash
npm install
```

2. **Start the development server**

```bash
npm run dev
```

3. Open the URL shown in your terminal (usually `http://localhost:5173`) to see the LMS UI.

## What You Get

- **RBAC-focused UI** showing Super Admin, Admin/HR, Trainer, and Learner roles.
- **Organization & Settings module** for:
  - Organization profile (name)
  - Branding color
  - Learning policies
  - Concurrent user limit
  - Security toggles (JWT/SSO, strong passwords)
- **Responsive, glassmorphism design** with gradients and soft shadows.

You can now plug this UI into your **Node.js/Express** backend and **MongoDB** database for real authentication, JWT handling, and persistence.
